description: 'Keep going? Resume a paused roadmap and continue pending tasks'
disable-model-invocation: true

Run the command below via `Bash` and follow the `content` field of the returned JSON:

- **macOS / Linux:** `.n45/bin/n45 init claude -c next`
- **Windows:** `".n45/bin/n45.exe" init claude -c next`

If `status: "error"` → show `message` to the user.
